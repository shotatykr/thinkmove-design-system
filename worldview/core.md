## 1. 全体を貫く「気分」

### 🎯 **「冷静な共感」**

静かな切迫感と、隣で聞いてくれる人の安心感が同居する空気。

- 派手なコピーで煽らない
- 派手なグラフィックで威圧しない
- でも行間には **「このまま止まったままでいいんですか？」** という静かな問いかけが流れている
- 叫ばないのに届く設計 — 「大声で売る広告会社」へのアンチテーゼ

**感情に寄り添うが、感情で売らない。**

## 2. コアレトリック（反復される3つの構文）

### 2.1 否定の後に差し出す

- 「**採用する前に。外注する前に。**」
- 「**上でも下でもなく、横で一緒に動く**」
- 「**代わりにやる人ではなく、一緒に前に進める人**」
- 「**答えを渡す支援ではなく、問いを共につくるパートナー**」

既存の選択肢（採用／外注／上司／部下／納品）を一度**否定**してから、その間に立つ**第三の選択肢**として自分を置く。異常なほど徹底された構文。

### 2.2 動詞を止める（そして動かす）

- 「**施策は回っている。でも、マーケティングは、どこかで止まっている。**」
- 「**誰にも責められていないのに、進められない。**」
- 「**止まっていた時間が動き出す。**」

「止まる」「詰まる」「進まない」という動詞が頻出。提供価値は常に **「動き出させること」**。社名 ThinkMove（Think & Move）がここに直結。

### 2.3 関係性で価値を定義する

- 「稼働時間を買う」のではなく **「判断が進むことを買う」**
- 「価格差は、会議回数ではなく、**文脈をどこまで持って伴走するか**の差」
- **「関与の深さ」** が提供価値の単位

## 3. 想定読者 — 過去の自分へのラブレター

### ペルソナ

> **「1〜2人体制の事業会社マーケ担当。経営と現場の間に挟まれて、Slackを開いても相談できる相手がいない人。」**

- 「**1人で詰まっていた過去の自分へ**」というコピーの通り、創業者自身の過去の自分へのラブレターとして設計
- 解像度が異常に高く、読み手は「これ、私のために書いてある」と感じる

### 誰を助けないかを明言する

- 「3名以上の複数人チームは支援対象外」
- **「誰を助けないか」を明言することで、「誰を助けるか」の輪郭**が鋭くなる

## 4. 差別化の置き方 — 第四の場所

既存の3カテゴリから**等距離に離れた第四の場所**に自社を置く。

| 既存カテゴリ | 欠点 |
|------------|------|
| 採用 | 文脈は溜まるが立ち上がりが遅く固定費が重い |
| 制作会社・代理店 | スコープ固定で柔軟性がない |
| コンサル | 叩き台を作らない、高い |
| **ThinkMove** | **文脈を持ち、即週動き、叩き台も作り、月20〜40万** |

**「既存カテゴリの欠点を全部拾って、全部解決する」** — カテゴリー・デザインの意識的な実装。

---

## 5. 価値の物差しの張り替え

3段階で物差しを書き換える。これが成立すると**価格競争から抜け出せる**。

| 通常 | ThinkMove |
|------|-----------|
| 数字の成果（PV・CV・順位） | **判断の質** |
| 稼働量（時間） | **関与の深さ** |
| 納品物 | **文脈の蓄積** |

- 「**レポートよりも、判断がほしい**」（ブログタイトル）
- 「**時間ではなく、関与の深さで考える**」（料金ページ）

**サイトの構造そのものがポジショニング戦略の実装**になっている。

---

## 6. 語り手の人格

### **「高い専門性を持ちながら、威圧しないプロ」**

- 営業・コンサル・事業責任者を全部やってきた（プロフィールで明示）
- 「**正しい施策が通らない組織の壁**」と「**何から手をつけていいかわからない現場の焦り**」の両方を知っている
- 偉そうにしない。煽らない。断定しすぎない（「〜かもしれません」の多用）
- でも踏み込むべきところは踏み込む（「3名以上は対象外」「ひとりで抱え込むな」）

**専門性は高いが偉そうじゃない、熱量あるけど押し付けない。**

## 7. 言葉選びの癖 — 造語戦略

既存のマーケ業界用語を避け、**自社の語彙で現象を指し示す**。これが「ThinkMoveの世界でしか通じない言葉」を作り、世界観の**閉域性・独自性**を生む。

| 造語・言い換え | なぜこの言葉か |
|----|------|
| **共創型R&D** | 単なる「支援」ではなく、研究開発性を帯びた営み |
| **問いの整流** | 整理ではなく、流れを整える |
| **インハウスマーケティング共創支援** | ありそうでない、独自の括り |
| **判断の型** | ノウハウやテンプレでなく「型」 |
| **文脈の蓄積** | 比較表での差別化の軸 |
| **同期セッション** | MTGや定例ではなく、非同期との対比で |
| **叩き台の作成** | 成果物ではなく「叩き台」— 思い切り |

新しい造語が定着したらここに追記する。

## 8. 視覚言語の思想

数値は [`theme/globals.css`](../theme/globals.css) と [`theme/design.json`](../theme/design.json) を正とする。ここでは**役割の意味**のみ定義。

### 8.1 配色

- **ネイビー基調**（落ち着き、信頼、夜の思考時間）
- **白の余白が多い** — 情報を詰め込まない。**「余白の主権」を自社サイトで実践**
- **Teal** = CTA 専任（前進、共創）
- **Orange** = 限定ハイライト（広面では使わない、CTAには絶対使わない）
- 比率の目安: 白70% / Navy 15% / Gray 10% / Teal 4% / Orange 1%

### 8.2 イラスト（Gemini 生成）

- 柔らかい線と淡い色合いのシーン描写（オフィスの隣に座る人、デスクで悩む人、4コマ漫画）
- 擬人化・感情表現は控えめ
- **「誰かの日常の一コマ」** として差し込まれる

### 8.3 写真 — 属人性を前に出す

- **豊藏さん本人のオフィス背景ポートレート**が会社の顔として繰り返し登場
- 属人性を隠さず、むしろ前に出す
- **「この人がやっている会社」という透明性が軸**

### 8.4 4コマ漫画 — 軽い器に入れる

- トップページ「こんな経験ありませんか？」セクション
- シリアスな課題を、漫画という軽い器に入れる
- **「重たい話を、重たく伝えない」** というブランドスタンス

### 8.5 トーンの基準

派手さはないが、安っぽくもない。

> **「良いコンサル資料のような清潔感」**

これがデザインの基準。

### 8.6 形状・シャドウ・モーション（思想のみ）

- **角丸（8〜20px、4段階）**: 親しみやすさ
- **shadow-as-border（Vercel流）**: 境界は太い線ではなく、微かな浮きで示す
- **動きは意味があるときだけ**
  - 許可: フェードイン／フェードアップ／ホバー浮き／カウントアップ
  - 禁止: パララックス／自動再生／点滅／バウンス／派手なローディング

## 9. 世界観を一文に圧縮すると

> # **「判断が止まっている場所に、隣に座って、問いを一緒に整える人」**

これが世界観の中心。サイトのあらゆる要素（コピー・イラスト・料金設計・比較表・FAQ）が、この一文を**別の角度から説明し直している**構造になっている。

## 10. 検証基準

### 10.1 言葉の検証

1. **否定→差し出しの構文**があるか
2. **「止まっている／動き出す」**のメタファーが機能しているか
3. **「関与の深さ」「文脈の蓄積」**など関係性で価値を定義しているか
4. **造語**を使って既存業界用語から逃げているか
5. **AI臭**がない（「〜と言えるでしょう」「〜が重要です」がゼロ）
6. **著者特定**できる（匿名でも豊藏さんが書いたと分かる）

### 10.2 視覚の検証

1. 3秒で「静かな信頼」が伝わる
2. 「落ち着く」「この人なら」と感じる
3. 他のコンサル会社・SaaSと違うと感じる
4. 閉じた後も「静かで、でも切迫感があった」と覚えている

## 11. 今後の課題（世界観として弱い部分）

正直に抱えている課題。v3 時点で認識している**未解決領域**。

### 11.1 「AI」の扱いが抽象的

- サイト全体で「AI×SEO」を標榜しているが、**AIが具体的に何をするかの描写が薄い**
- 「生成AIの使える設計」「プロンプト整備」程度
- AI側の世界観はまだ言語化途中

### 11.2 事例の深さに差がある

- Makuake・HowMa は掘れている
- 他クライアントはロゴだけ並ぶ状態
- ここで世界観の「手触り」が薄くなる

### 11.3 LP と会社概要の温度差

- **LP（熱）× 会社概要（冷）** で文体が二層化
- 「Think & Move」「Issue Driven Thinking」等の5つのバリューは、LPの尖った世界観と比べるとフォーマルで温度差
- 会社概要側を LP の温度に寄せたい
