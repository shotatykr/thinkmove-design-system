# Quote Card

## 構造

```
        ─────────────────
        ─────────────────

         [中央に大きく]
       「 共 創 型 R & D 」

        ─────────────────
        ─────────────────

       (下に小さく1行の補足)
```

## tokens

| 要素 | 値 |
|------|----|
| 背景 | `color.bg-alt` または `color.primary-light` |
| 引用符（上下） | `color.cta` の細い線、80px幅、中央 |
| 造語フォントサイズ | 既存 H1 の 1.5倍程度 |
| 造語色 | `color.ink` |
| 造語 letter-spacing | 0.05em（CJK 余白） |
| 補足行 | `color.text-secondary`、font-size 14px |
| 余白 | 上下35%、左右20% |

## 良い例

> 　
> 　
>
> ## 「 共 創 型 R&D 」
>
> 単なる支援ではなく、研究開発性を帯びた営み
> 　
> 　

## 悪い例

- ❌ 造語を **太字＋ハイライトマーカー＋アイコン** で装飾過多
- ❌ 文中に造語を埋める（埋もれて伝わらない、quote-card の意味がない）
- ❌ 1スライドに造語2個以上（focus分散）
