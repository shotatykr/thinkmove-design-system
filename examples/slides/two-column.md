# Two-Column

## 構造

```
┌────────────────────┬────────────────────┐
│  LEFT (止まる側)    │  RIGHT (動く側)    │
│                    │                    │
│  H3: 状態の名前     │  H3: 状態の名前    │
│  ─────────────     │  ─────────────    │
│  • 症状1            │  • 解1             │
│  • 症状2            │  • 解2             │
│  • 症状3            │  • 解3             │
│                    │                    │
│  灰色トーン          │  navy/teal トーン   │
└────────────────────┴────────────────────┘
```

## tokens

| 要素 | 値 |
|------|----|
| 左カラム背景 | `color.bg-alt` |
| 左カラム H3 | `color.text-secondary`（沈める） |
| 右カラム背景 | `color.bg`（白） or `color.primary-light` |
| 右カラム H3 | `color.primary` または `color.cta-dark` |
| カード radius | `radius.lg` (16px) |
| カラム間 gap | 24-32px |
| shadow | `shadow.sm` 各カラムに |

## 「止まる/動く」のメタファーマッピング

| 左（止まる） | 右（動く） |
|------------|----------|
| 採用 | ThinkMove |
| 外注（スコープ固定） | 共創（柔軟） |
| 稼働時間 | 関与の深さ |
| 納品物 | 文脈の蓄積 |
| 数字の成果 | 判断の質 |

## 良い例

| 採用 | ThinkMove |
|------|-----------|
| 立ち上がりが遅い | 即週動く |
| 固定費が重い | 月20〜40万 |
| 文脈は溜まる | 文脈を持ち、叩き台も作る |

## 悪い例

- ❌ 左右どちらも同じトーン（対比が弱い）
- ❌ 右に派手な ✨ や 🚀 アイコン（「冷静な共感」と齟齬）
- ❌ 4カラム以上（情報密度過多、worldview の「白の余白」と矛盾）
