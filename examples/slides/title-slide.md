# Title Slide

## 構造

| 要素 | 内容 |
|------|------|
| H1（中央） | **結論を1行で断言**。句点なし、25字以内 |
| サブタイトル | **問い形式**（「〜のままでいいか？」「〜していませんか？」） |
| フッター（小） | 章番号 or 造語1個 or 日付 |

## tokens（`../../theme/design.json` 参照）

| 要素 | 値 |
|------|----|
| 背景 | `color.bg`（白）または `color.ink`（dark反転） |
| H1 文字色 | `color.text`（white背景時）or `color.bg`（dark背景時） |
| H1 font-weight | 700、letter-spacing -0.01em |
| 余白 | 上下40%、左右15%（中央寄せ） |
| サブタイトル色 | `color.text-body` |
| 装飾 | なし（白背景の場合）or `color.cta` の細い下線（dark背景の場合） |

## 良い例

> # 判断が止まっている場所に、隣に座る
> 採用する前に。外注する前に。

> # 文脈の蓄積が、関与の深さを決める
> 稼働時間を買っていませんか？

## 悪い例

- ❌ 「AI時代のSEO戦略について」 — 動詞が止まっている、断言していない
- ❌ 「3つのポイントで分かるマーケティング」 — N個の理由型、AI臭
- ❌ 装飾的アイコンを並べる、グラデーション多用 — 「冷静な共感」と齟齬
